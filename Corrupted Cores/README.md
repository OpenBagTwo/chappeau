# Corrupted Cores

This pack lets you wear the corrupted cores from _Portal 2_ on your head.

## To Use

Grab a carved pumpkin and rename it in an anvil with one of the following names:
- Aquarium Core
- Fact Sphere
- Morgan Freeman Core
- Rick
- Space Core
- Wheatley
