# Spheres with Leaves

This pack adds Oddishes of various flavors.

## To Use

Grab one of the following items and name it "Oddish"

| Base Item | Color |
| --- | --- |
| Short Grass | Green |
| Seagrass | Yellow |
| Beetroot | Red |
